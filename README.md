# DEFINE-INCLUDE-FOR-c-CPP
# 各个语言编辑所需的引入和声明

## C++

```cpp
\#include <iostream> // 标准输入输出流
\#include <vector>   // 标准模板库中的向量
\#include <string>   // C++ 字符串类
\#include <cmath>    // 数学函数
using namespace std;

int main() {
std::cout << "Hello, world!" << std::endl;
return 0;
}
```

## C

```c
#define  _CRT_SECURE_NO_WARNINGS
#include <iostream> // 标准输入输出流
#include <vector>   // 标准模板库中的向量
#include <string>   // C++ 字符串类
#include <cmath>    // 数学函数

int main() {
    std::cout << "Hello, world!" << std::endl;
    return 0;
}

```
